NAME

infopkg - Display detailed package information for a given package file.  

SYNOPSIS

infopkg [ -h, --help ] [ -m, --menu ] pkgfile [ pkgfile2, ... ]  

DESCRIPTION

infopkg displays detailed information about a package file, outputting either to the console or a dialog-style information-box. This can be used to verify that a package build/make succeeded, or that a given package file doesn't contain malicious files.  

OPTIONS

-h, --help
When this option is specified, infopkg will print its own usage instructions and exit.
-m, --menu
This option will cause infopkg to display the given package's information inside a scrolling dialog, instead of printing it to the console.
