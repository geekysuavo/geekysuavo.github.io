NAME

explodepkg - Extract the contents of a system package to the current working directory.  

SYNOPSIS

explodepkg [ -h, --help ] [ -v, --verbose ] pkgfile [ pkgfile2, ... ]  

DESCRIPTION

explodepkg uncompresses and untars pkg-scripts compatible packages into a subdirectory in the current working directory.  

OPTIONS

-h, --help
When this option is specified, explodepkg will print its own usage instructions and exit.
-v, --verbose
This option will cause explodepkg to output more information about the build process than usual. In particular, this will print every exploded package as it is being unpacked.
